Declaración de una función

Se le asigna un nombre a la función. Dicha función se debe cargar en el ámbito de la aplicación antes de la ejecución del código. 

          
            function miFuncion() {
              console.log("Esta es una función clásica");
            }
            
            miFuncion(); // Salida: Esta es una función clásica

Expresiones function

Se crea una función anónima y se le asigna a una variable. Esta función no se encuentra lista hasta que esa línea en particular sea evaluada durante la ejecución del código 

          
            let smartphone = function (marca) {
              console.log(`Marca smartphone: ${marca}`);
            };
            
            smartphone("Sony"); // Salida: Marca smartphone: Sony

Arrow function

Sirve para simplificar la escritura del código. Se pasa el nombre de la función para detrás del paréntesis y se coloca el símbolo =>.
En sus dos maneras la función pierde el nombre y se manda solo el parámetro si es uno solo. Puede ir sin paréntesis. 

          
            let smartTV = (marca) => {
              return `El mejor Smart TV es ${marca}`;
            };
            
            console.log(smartTV("El mejor Smart TV es Samsung"));

            


            let miniComponente = (marca) =>
              console.log(`El mejor minicomponente es ${marca}`);
          
            miniComponente("Sony"); // Salida: El mejor minicomponente es Sony

Con más de un parámetro:

          
            let procesadores = (marca1, marca2) =>
              console.log(
                `Los procesadores que dominan el mercado son de ${marca1} y ${marca2}`
              );

            procesadores("Intel", "AMD"); // Salida: Los procesadores que dominan el mercado son de Intel y AMD
