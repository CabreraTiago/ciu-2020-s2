Var

  • Las variables declaradas con var son procesadas antes de la ejecución del código.
  • El scope de una variable declarada con var, es su contexto de ejecución.
  • El scope de una variable declarada fuera de la función es global.

Let

La instrucción let declara una variable de alcance local con ámbito de bloque(block scope), la cual, opcionalmente, puede ser inicializada con algún valor.

Const

Las variables constantes presentan un ámbito de bloque (block scope) tal y como lo hacen las variables definidas usando la instrucción let, con la particularidad de que el valor de una constante no puede cambiarse a través de la reasignación. Las constantes no se pueden redeclarar.

Ejemplos

          
            var desktop = "Lenovo";
            let notebook = "Dell";
            const allInOne = "HP";
            
            if (desktop) {
              var desktop = "Asus";
              console.log(`Dentro del condicional: ${desktop}`); // Salida: Dentro del condicional: Asus
            }
            console.log(`Fuera del condicional: ${desktop}`); // Salida: Fuera del condicional: Asus
            
            if (notebook) {
              let notebook = "Acer";
              console.log(`Dentro del condicional: ${notebook}`); // Salida: Dentro del condicional: Acer
            }
            console.log(`Fuera del condicional: ${notebook}`); // Salida: Fuera del condicional: Dell
            
            console.log(`Constante: ${allInOne}`); // Salida: HP
