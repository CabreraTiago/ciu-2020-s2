Object literal

El Object literal nos permite usar un valor por cada variable. Si necesito un objeto distinto pero con la misma estructura, tengo que crear otro objeto. 

          
            const notebook = {
              marca: "Dell",
              modelo: "Latitude 5480",
            };

Acceso a los valores de un objeto

          
            console.log(notebook.marca); // Salida: Dell
            console.log(notebook.modelo); // Salida: Latitude 5480
            
            console.log(notebook["marca"]); // Salida: Dell

Object Constructor

Si necesitamos muchos objetos del mismo tipo tenemos que usar Object Constructor. 

          
            function Tablet(marca, modelo, pantalla) {
              this.marca = marca;
              this.modelo = modelo;
              this.pantalla = pantalla;
            }
            
            const tabletApple = new Tablet("Apple", "Air", 9.7);
            const tabletLenovo = new Tablet("Lenovo", "M10", 10.1);
            
            console.log(tabletApple); // Salida: Tablet { marca: 'Apple', modelo: 'Air', pantalla: 9.7 }
            console.log(tabletLenovo); // Salida: Tablet { marca: 'Lenovo', modelo: 'M10', pantalla: 10.1 }

Prototipos

Los prototipos nos sirven para "atar" una función a un determinado tipo de objeto, como si fuera un método propio de ese objeto. 

          
            Tablet.prototype.describirTablet = function () {
              return `La tablet ${this.modelo} de ${this.marca} tiene ${this.pantalla} pulgadas`;
            };
            
            console.log(tabletApple.describirTablet()); // Salida: La tablet Air de Apple tiene 9.7 pulgadas
            console.log(tabletLenovo.describirTablet()); // Salida: La tablet M10 de Lenovo tiene 10.1 pulgadas

Object Destructuring

La sintaxis de desestructuración es una expresión de JavaScript que permite cargar valores de arreglos o propiedades de objetos en distintas variables. 

          
            const smartphones = {
              iphone: {
                entrada: "iPhone 12",
                media: "iPhone 12 Pro",
                alta: "iPhone 12 Pro Max",
              },
              otros: ["Samsung S20", "Huawei P30", "Asus Zenfone 7"],
            };
            
            let { entrada } = smartphones.iphone;
            console.log(`Usando Object Destructuring: ${entrada}`); // Salida: Usando Object Destructuring: iPhone 12

Object literal enhacement

Se utiliza para agrupar variables del ámbito global y convertirlas en objetos. Es el proceso de reestructuración o reconstrucción. 

          
            let marca = "Asus";
            let modelo = "Zenfone 7";
            let caracteristicas = [
              "Procesador Snapdragon 865+",
              "8GB RAM",
              "128GB Almacenamiento",
            ];

            const zenfone = { marca, modelo, caracteristicas };

            console.log(zenfone);

            // Salida:
            // {
            //   marca: 'Asus',
            //   modelo: 'Zenfone 7',
            //   caracteristicas: [ 'Procesador Snapdragon 865+', '8GB RAM', '128GB Almacenamiento' ]
            // }

Agregando funciones a objetos

Cuando creamos un objeto, también le puedo agregar funciones propias en la descripción. Usando la palabra reservada this no tengo que pasar parámetros a esta función, ya que los toma de la descripción del mismo. 

          
            const xboxOneS = {
              marca: "Microsoft",
              modelo: "Xbox One",
              procesador: "AMD Jaguar",
              tipo: "digital",
              caracteristicas() {
                console.log(
                  `La ${this.modelo} de ${this.marca} tiene un procesador ${this.procesador} y es completamente ${this.tipo}.`
                );
              },
            };
            
            xboxOneS.caracteristicas(); // Salida: La Xbox One de Microsoft tiene un procesador AMD Jaguar y es completamente digital.
